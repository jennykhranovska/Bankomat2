# Bankomat

Det här är ett konsolprogram som simulerar en bankomat med PIN-kod, meny och funktioner för insättning, uttag och saldo.

## Funktioner

- Inloggning med PIN-kod
- Meny för att välja insättning, uttag, visa saldo eller avsluta
- Inkapsling av saldo med privata fält och publika metoder
- Hantering av giltiga och ogiltiga belopp

## Hur man kör programmet

1. Klona eller ladda ner projektet från Git.
2. Öppna projektet i Visual Studio.
3. Kör programmet (tryck F5 eller Ctrl+F5).
4. Följ instruktionerna i konsolen för att logga in och använda bankomaten.
